# Tallos
Sistema de gerenciamento de funcionários

# Como rodar a aplicação?

Faça download <br>
git clone https://github.com/dev-MarcosVinicius/Tallos.git

Entre no diretorio do servidor<br>
cd server/

Instale os node_modules<br>
npm install

# Executar o Front-End
cd tallossys/ <br>
npm run serve

# Executar o Back-End
cd server/ <br>
npm run start
